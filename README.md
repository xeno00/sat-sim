# sat-sim

Satellite simulation and analysis code for developing manuscript results for the
JCLS non-terrestrial network study.

## Purpose

This repository is intended to hold reproducible simulation code, data-generation
utilities, and analysis scripts used to support manuscript figures and numerical
experiments.

## Initial Scope

- Satellite and user-equipment geometry generation
- Link and measurement simulation utilities
- Localization and synchronization algorithm experiments
- Figure-generation and result-validation scripts

## Status

This repository has been initialized for code development. Implementation modules
and reproducibility instructions will be added as the simulation workflow is
developed.
